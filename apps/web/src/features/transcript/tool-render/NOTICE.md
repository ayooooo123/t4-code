# OMP tool renderer provenance

The renderer core in this directory (`types.ts`, `util.ts`, `parts.tsx`,
`generic.tsx`, `registry.ts`, and `tools/*.tsx`) is adapted from the
host-agnostic Oh My Pi Collab renderer at commit
`d7c9ac81a3764085d050d0b7148ac7eee976ddd3`.

Source: `packages/collab-web/src/tool-render` in Oh My Pi.

Oh My Pi is distributed under the MIT License. The applicable copyright and
license text is retained in `licenses/OMP-COLLAB-TOOL-RENDERERS-MIT.txt`.

T4-specific integration, result-shape compatibility, semantic-token styling,
and transcript-image handling are maintained in this repository.
